# Netflix Clone 🎬

A modern Netflix UI clone built with React.js and Tailwind CSS. This project replicates the core Netflix user experience, including dynamic category carousels (using Swiper.js), personalized "Continue Watching" progress bars, and a specialized "Top 10" ranking view.


## ✨ Features
* **Dynamic Carousels:** Responsive movie sliders powered by Swiper.js.
* **Category Filtering:** Logic-based rendering for "Top 10", "Trending", and "Watching" categories.
* **Progress Tracking:** Custom-built progress bars for the "Continue Watching" section.
* **Data-Driven UI:** Decoupled data logic (moviesData.js) for easy content management.

## 🛠️ Tech Stack
* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **Slider Library:** Swiper.js
* **Bundler:** Vite
